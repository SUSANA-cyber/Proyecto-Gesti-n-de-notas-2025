Avance del proyecto 04 Eliminación de cursos

- El programa permite eliminar cursos de la lista, de cursos registrados. 
- Al seleccionar la opción eliminar 
- Le pide al usuario que ingrese el nombre del curso que desea eliminar 
- Después de que el usuario ingrese el nombre del curso le pide la confirmación para eliminar el curso.

- Si el usuario ingresa si se elimina el curso, y si el usuario ingresa no se cancela la eliminación del curso.

- Si el usuario ingreso si se elimina el curso y la nota que tiene
- Al finalizar aparece un mensaje CURSO ELIMINADO CORRECTAMENTE

- Si el usuario ingresa no se cancela la eliminación del curso y aparece un mensaje ELIMINACION CANCELADA
